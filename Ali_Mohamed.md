What SSH is used for

Developers and system admins use SSH for things like:

 Accessing a remote server

 Transferring files securely

 Running commands on another machine

 Deploying applications

 Authenticating with keys instead of passwords

Example:
If you have a Linux server on the internet, you can control it from your computer with SSH.

/////////////////////////////////////////////////////

Example SSH command
ssh username@server_ip

Example:

ssh root@192.168.1.10

This means:

connect to server 192.168.1.10

login as user root

After that, you get a terminal of that remote machine.
